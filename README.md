## Установка:

Рекомендуется использовать PyCharm

Ссылка для добавления проекта
[git@github.com:viktorefimovich/online_store.git]()

Для создания виртуального окружения и установки зависимостей используйте `Poetry`. Установка `poetry install`.

`_Раздел дорабатывается_`

## Использование:

Проект использует фреймворк `django`. Установка `poetry add django`.

Запустите локальный http-сервер командой `python3 manage.py runserver`

Зайдите в браузере по ссылке [127.0.0.1:8000]()
